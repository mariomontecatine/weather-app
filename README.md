# Real-Time Weather App

A basic real-time weather application that retrieves data from the Visual Crossing API.  
It displays current temperature, timezone, latitude and longitude, weather alerts, and an icon that matches the weather conditions.

## Features
- Fetches real-time weather data from an external API  
- Shows key weather information  
- Uses Tufte CSS for layout and typography  

## Notes
This project was created as an initial exploration of working with APIs and Tufte CSS. *Webpack was not used due to existing npm vulnerabilities*.
